SchoolSoft Calendar Helper hjälper elever och föräldrar att hitta prov, inlämningar, redovisningar och andra viktiga datum utan att behöva leta runt i SchoolSoft.

Integritet

SchoolSoft-data stannar lokalt.

Som appen distribueras här körs den på din dator. Kalenderdata, anteckningar och sparad webbläsarsession lagras lokalt på datorn.

  • Ingen central server tar emot SchoolSoft-data.
  • Inloggning sker på SchoolSofts egen sida.
  • Appen är inte en officiell SchoolSoft-produkt.
  • GitHub Pages kan hantera vanlig webbplatsloggning för själva nedladdningssidan.

Frågor

Windows visar “Windows protected your PC”. Hur startar jag ändå?

Klicka More info / Mer information. Då brukar knappen Run anyway / Kör ändå visas. Klicka på den för att fortsätta. Varningen visas eftersom appen är ny och osignerad, inte nödvändigtvis för att Windows har hittat skadlig kod.

Varför visas SmartScreen-varningen?

Appen är inte kodsignerad ännu och har ingen etablerad Microsoft SmartScreen-reputation. För en bredare publik behöver appen signeras med ett kodsigneringscertifikat.

Varför installeras appen inte helt automatiskt i webbläsaren?

Webbläsare får inte starta och installera program helt tyst. Knappen laddar ned installationsfilen. Därefter behöver du öppna filen och godkänna installationen på datorn.

Fungerar det på alla datorer?

Målet är Windows, macOS och Linux. Varje plattform behöver en egen byggd fil. De första versionerna behöver testas på rena datorer. macOS och Windows kan visa säkerhetsvarningar tills appen är kodsignerad.

Är detta en officiell SchoolSoft-app?

Nej. Det är ett inofficiellt, öppet verktyg för lokal användning av data du själv är behörig att se.

Vad händer om SchoolSoft ligger nere?

Appen kan visa tidigare sparad lokal data, men kan inte hämta ny data förrän SchoolSoft fungerar igen.
