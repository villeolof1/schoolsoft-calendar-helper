Integritet

Lokal först.

Som appen distribueras här körs den på din dator. SchoolSoft-data, anteckningar och sparad webbläsarsession lagras lokalt på datorn.

  • Ingen central server tar emot SchoolSoft-data.
  • Inloggning sker på SchoolSofts egen sida.
  • Appen är inte en officiell SchoolSoft-produkt.
  • GitHub Pages kan hantera vanlig webbplatsloggning, till exempel IP-adresser, för själva nedladdningssidan.

Frågor

Varför installeras den inte helt automatiskt i webbläsaren?

Webbläsare får inte starta och installera program helt tyst. Knappen laddar ned installern; därefter behöver du öppna filen och godkänna installationen i Windows.

Är detta en officiell SchoolSoft-app?

Nej. Det är ett inofficiellt, öppet verktyg för lokal användning av data du själv är behörig att se.

Varför kan Windows varna vid installation?

Appen är inte kodsignerad ännu. Därför kan Windows SmartScreen visa en varning trots att källkoden är öppen.

Vad händer om SchoolSoft ligger nere?

Appen kan fortfarande visa tidigare sparad lokal data, men kan inte hämta ny data förrän SchoolSoft fungerar igen.
